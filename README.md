# LIN Cloud Postgres SQL Manager

LIN Cloud Postgres Manager es una herramienta que permite gestionar bases de datos de un mismo servidor, de forma sencilla y rápida.
Permite crear, editar y eliminar bases de datos y asignarlas a un usuario concreto, permitiendo subarrendar bases de datos a otros usuarios.

### Estado actual.
Este proyecto ya no se mantiene, y fue reemplazado por LIN Cloud Edge (Lake) que permite crear espacios aislados para cada recurso de LIN Cloud.
